# 스프링 웹 프로젝트 - 그룹웨어 팀웍스(Teamworks)

스프링 부트 프레임워크를 이용한 그룹웨어 사이트 제작

팀원 : 백인호, 안민현, 임다솔, 정민희

프로젝트 기간: 2021-07-12 ~ 2021-08-25

<br>

## 개발 환경

- 웹 프레임워크: Spring Boot
- DB: Oracle, Mybatis
- 템플릿 엔진: JSP
- 로그인, 권한 처리: Spring Security
- 메일: improvmx(Email forwarding), cafe24(DNS)
- 배포: AWS (EC2, RDS)

<br>
